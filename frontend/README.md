# Page Pilot — Frontend

React 19 + Vite frontend for the Page Pilot RAG chatbot.

## Dev server

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173` — make sure the backend is also running on port `8000`.

## Build

```bash
npm run build
```

## Stack

- **React 19** with hooks (useState, useEffect, useRef)
- **Vite** for fast builds and HMR
- **Vanilla CSS** — no UI framework, fully custom design
